# Scrubbit — Desktop Downloads

Privacy-first document anonymization for Mac and Windows.

## Download

Go to the [Releases page](https://github.com/ruth989/scrubbit-releases/releases) to download the latest version.

- **Mac (Apple Silicon)**: Download the `.dmg` file
- **Windows**: Download the `.exe` installer

### Installation Notes

- **macOS**: You may need to right-click → Open on first launch (the app is not code-signed yet)
- **Windows**: SmartScreen may warn "Unknown publisher" — click "More info" → "Run anyway"
